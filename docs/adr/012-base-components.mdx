
# Base Component Framework

* Status: proposed
* Deciders: Jude, Nazim, Florian
* Date: 31.07.2023

Technical Story: [description | ticket/issue URL]


## Considered Options

* https://mui.com/base-ui
* https://zagjs.com/
* https://react-spectrum.adobe.com/react-aria/index.html
* https://www.radix-ui.com/

## Decision Outcome

Chosen option: "mui.com/base-ui", because it's widely used, has a big community, is extensible and focuses on using hooks.
